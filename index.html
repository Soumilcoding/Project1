Python

Les appareils qu'on utilise, les logiciels, les ordinateurs, utilisent un langage très spécifique ! Dans ce cas, le codage. C'est des instructions qu'on donne à un ordinateur afin d'exécuter une certaine tâche.  Il ne faut pas faire d'omission, car tout ce que l'ordinateur lit doit être précis, sinon, il y aura une erreur et le code ne pourra pas marcher. À la nature du codage, il y a quelque chose qu'on tente de résoudre, par exemple : comment créer un logiciel d'addition ? On met les idées en premier, puis on procède en écrivant le code. Il peut parfois y avoir des erreurs, mais on peut corriger ce qu'on trouve incorrect et relancer le code. Enfin, lorsque le code de base marche, on peut le raffiner en ajoutant des éléments supplémentaires ! Dans ce cas, d'autres fonctions mathématiques telles que la multiplication.

Afin de pouvoir utiliser Python, on peut accéder à un runner qui court le code. Dans un site HTML, il est possible d'importer certains modules pour pouvoir utiliser Python ! Le processus entier est un peu compliqué. Mais cela peut permettre l'obtention de boites de textes comme celle en dessous!

Il y a des manières d'utiliser Python et d'expérimenter avec. Celui que j'utilise personnellement est Replit ! Replit permet facilement de courir un programme Python sans avoir à faire autre chose ! En addition, on ne doit pas préparer des lignes de code en avance, on peut directement coder  sur ce site ! Pour un site web HTML tel que celui que j'utilise présentement, je dois ajouter une section de plus dans la tête de mon code. Je me dois aussi d'importer des modules. Ensuite, cela crée une fenêtre ou un espace dans le site qui court le code Python.

HTML

HTML est le langage utilisé afin de créer ce site. Si Python est pour certaines fonctions, par exemple des opérations simples, alors HTML est pour le site en soi ! En fait, il est très simple à utiliser. Pour le texte écrit jusqu'à présent, utilise le <p> </p>. Je mets ce que je veux dire ENTRE ces lettres. C'est le format pour les sites web le plus recommandé pour sa facilité d'utilisation et la possibilité de le styliser avec d'autres composantes, telles que CSS. CSS étant un module qui change la couleur, la police, la taille, etc.
